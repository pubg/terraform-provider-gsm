# gsm_game_server_cluster (Data Source)

Use this data source to retrieve configuration details of an existing game server cluster.

~> **Note:** This data source does not include sensitive fields like private keys and JWT tokens. Use the resource configuration directly if you need those values.

## Example Usage

```terraform
data "gsm_game_server_cluster" "example" {
  cluster_name = "my-cluster"
}

# Use the data source outputs
output "cluster_kube_issuer" {
  value = data.gsm_game_server_cluster.example.kube_issuer
}

output "cluster_agones_endpoint" {
  value = data.gsm_game_server_cluster.example.agones_endpoint
}
```
